Introduction Image Tag

in this page use the image tage for display the image img tag is useful for display the image in the web page this tage contains different attributes

Attribute of img Tag

  • src

    • source file path

  • alt

    • alternative if image not found or load the image slow then it display the text

  • width

    • width of the image

  • height

    • height of the image

Syntax of display image in web page

                <img src="image_path" alt="text" width="image_width" height="image_height" />
